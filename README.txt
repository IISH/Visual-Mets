How to use

- Install a tomcat server
- Install you METS war file on the tomcat server
- Store your files on a fileserver, accessible over the HTTP protocol
Preferably you can link to your objects by an persistent identifier

- Create METS documents <link naar voorbeeld METS>
Embed links to:
- Thumbnail (max size <invullen>)
- Reference image (max size <invullen>)
- Optional: transcription (in text format)
- Store your METS files on a HTTP accessible location, otherwise the API cannot read the METS documents
- Embed the javascript on your website to show the METS viewer
